Task 3: AWS + Vercel Showcase

Cloud Infrastructure Task Assignment

Objective

You will create basic infrastructure resources in AWS across multiple services. After completing each, you'll take screenshots as proof of completion. Then, you'll design a simple webpage using Vercel to display your screenshots.

Tools Used:
  • Terraform or AWS Console (for infrastructure)
  • Vercel (for webpage)
  • Google Form (for submission)

Part 1: AWS Resources to Create (Free Tier Only)

For each service below, create the simplest working setup. You may use the AWS Console or Terraform for provisioning.

AWS Service Task to Perform
Amazon EC2 Launch a t2.micro instance with Amazon Linux 2
Amazon VPC Create a custom VPC with 1 public subnet and Internet Gateway
AWS Lambda Create a simple Lambda function (e.g., Hello World in Python)
Amazon S3 Create a private bucket with versioning enabled
Amazon EBS Attach a 10GB EBS volume to your EC2 instance
Amazon RDS Launch a db.t2.micro RDS instance (e.g., MySQL)
Amazon Athena Create a database and table from sample S3 data
Amazon CloudWatch Create an alarm for EC2 CPU utilization > 70%
AWS CloudTrail Enable CloudTrail to monitor API activity
AWS CloudFormation Deploy a basic template to create an EC2 instance
AWS CDK Use CDK to provision an S3 bucket (in TypeScript or Python)
Terraform Tip: For services you create with Terraform, use this basic structure:
# Example EC2 configuration resource "aws_instance" "example" { ami = "ami-0c55b159cbfafe1f0" # Amazon Linux 2 instance_type = "t2.micro" tags = { Name = "Task3-Instance" } }
